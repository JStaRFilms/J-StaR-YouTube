# Anniversary Editorial (V3) - "The Stomp"

## Vision
A high-energy, typo-centric animation. Less "romantic/soft", more "bold/unbreakable". Think Nike ad meets a Love Story.

## Visuals
- **Palette**: Monochrome (Black/White) + Electric Purple (#8A2BE2) or Neon Lime. Let's go **Electric Purple**.
- **Fonts**: `Anton` (Massive Headers), `Inter` (Details).
- **Motion**: Snap zooms, sliding masks, kinetic typography.

## Breakdown
- [ ] **Scaffold**: `src/AnniversaryEditorial/` folder and `Root.tsx` registration.
- [ ] **Assets**: Load `Anton` font. Define new Color constants (Purple/Black/White).
- [ ] **Component: `BigType`**: A component for massive, screen-filling text that scales/slides.
- [ ] **Component: `PhotoGrid`**: Abstract CSS grid that simulates a rapid photo burst (using colored blocks if no photos).
- [ ] **Scene 1: The Intro**: "TWO SOULS" -> "ONE VIBE".
- [ ] **Scene 2: The Numbers**: "1365 DAYS" (or whatever number) counting up fast.
- [ ] **Scene 3: The Finale**: "JENNY" / "FOREVER".
